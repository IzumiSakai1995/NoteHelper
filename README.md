# 项目介绍

这是一个辅助网游类小说写作的项目，写一个简单的网络游戏，用于辅助小说作者进行游戏设计和游戏测试。

## 项目依赖

### 框架

- Tauri2.0：用于构建跨平台的桌面应用程序
- Vue3：用于构建前端用户界面

### 数据库

## 项目功能

- 游戏设计：作者可以在游戏中设计角色、场景、事件等，为小说添加更多的内容。
- 装备系统：作者可以在游戏中设计装备
- 怪物系统：作者可以在游戏中设计怪物
- 事件系统：作者可以在游戏中设计事件
- 地牢系统：作者可以在游戏中设计地牢，每个地牢可以绑定不同的怪物

## 管理

- 新增玩家
- 新增怪物
- 新增地牢
- 新增事件
- 新增地图
- 设置地牢里的怪物
- 设置事件里的怪物



## 字段设计

### 玩家

- 玩家名
- 玩家等级
- 玩家经验
- 玩家生命值
- 玩家攻击力，通过力量和敏捷或者智力计算得来
- 玩家防御力，通过体力和相关系数计算得来
- 力量，提供攻击力，少量血量
- 敏捷，提供更多的攻击力
- 智力，提供魔法攻击力
- 体力，提供血量和防御力
- 精神，提供魔法回复
- 玩家装备

### 怪物

- 怪物名
- 怪物等级
- 种族
- 稀有度
- 怪物前缀
- 生命值
- 攻击力
- 防御力
- 伤害减免
- 掉落物品
- 描述

### buff

- buff名
- buff类型
- buff持续时间
- buff效果

## 物品管理

- 新增物品,需要选择物品类型
- 设置物品属性

### 物品类型管理

- 增删改物品类型

## 装备管理

装备视为物品之一，继承自物品

- 新增装备,需要选择装备类型
- 设置装备属性

## 技能管理

- 增删改技能
- 角色绑定技能

## 数据展示

单独页面展示玩家、怪物、地牢、事件、地图、物品、技能、buff等数据。

## 伤害计算

单独页面选择玩家，选择怪物，计算伤害
- 需要展示玩家的具体属性
- 复选框添加buff效果
- 勾选是否暴击
- 勾选是否致命一击
